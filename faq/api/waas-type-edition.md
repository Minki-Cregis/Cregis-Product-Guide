# API 项目创建后还可以编辑类型吗？

可以，创建后的项目仍可编辑类型，请参考以下步骤进行操作：

1.  点击WaaS项目后进入 **项目设置 > 开发者**。<br>

2.  启用接口类型<br>

    <figure><img src="../../.gitbook/assets/image (659).png" alt=""><figcaption></figcaption></figure>

<figure><img src="https://0xcregis.sg.larksuite.com/space/api/box/stream/download/asynccode/?code=YmY3MTllMDc3NDU1MTIxNWM0YmJkMjA2MTk0N2MzZGZfSjE3a0pqa0g3SkVsMmcwbGN2SkFoakVqQ05ENkQwbnRfVG9rZW46WDhxVmJMRm1MbzFRNVV4SlA3Z2xQSXMxZ1VrXzE3NjA5NDU5ODA6MTc2MDk0OTU4MF9WNA" alt=""><figcaption></figcaption></figure>

3.  选择出款钱包（多选），并指定一个默认出款钱包，点击下方「**确定**」\
    提示：当API出款未指定出款钱包时，系统会使用设置的默认出款钱包的默认付款地址进行出款<br>
