# Why Can’t the Member See the Shard Import Entrance When I Try to Authorize a Shard?

To authorize a shard to a member, ensure that the wallet creator has added the member to the wallet. You can find the operation entry here:

<figure><img src="../../.gitbook/assets/image (284).png" alt=""><figcaption></figcaption></figure>

Once the member has been added to the wallet, you can proceed with the authorization. If the creator hasn’t completed the authorization process, the member <mark style="color:green;">**will not see the import option**</mark>. Detailed authorization steps can be found [here](../../wallet-and-transactions/shard/#authorize-a-shard-to-another-wallet-member).
