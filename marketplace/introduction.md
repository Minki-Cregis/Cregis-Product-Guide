# Introduction

The Application Marketplace is a core component of the wallet ecosystem, designed to aggregate various high-frequency Web3 extension services.

This module has now launched its first remittance application, allowing users to initiate transfer requests directly through their wallet assets. To build a richer ecosystem matrix, we will gradually onboard more security-verified partner providers. Through the Application Marketplace, users can enjoy a full range of digital asset services and a seamless interactive experience within a unified wallet environment, without needing to switch between multiple platforms.
